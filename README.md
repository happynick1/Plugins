# 🎨 语音助手插件仓库

这是语音助手（Happy AI）的官方插件仓库，提供丰富的插件来扩展和美化您的语音助手功能。

## 📦 可用插件

### 1. UI 美化插件 (`ui_theme_plugin.py`)

**功能介绍：**
- 🎨 自定义语音助手的配色方案
- 🎯 支持多种预设主题（默认蓝、科技蓝、商务黑、清新绿、浪漫紫、暖橙色、少女粉、薄荷青）
- 🖼️ 实时预览主题效果
- 💾 一键保存并应用主题
- 🔄 支持自定义颜色选择

**使用方法：**
1. 下载 `ui_theme_plugin.py` 文件
2. 将文件放入语音助手的 `tool` 目录
3. 重启语音助手
4. 进入"插件市场" → "已安装"
5. 点击 UI 美化插件卡片
6. 选择喜欢的主题，点击"立即应用"
7. 重启语音助手查看效果

**文件大小：** 8KB  
**版本：** 1.0.0

## 📖 插件开发规范

如果您想开发自己的插件，请遵循以下规范：

### 基本结构
#### python 
```

def plugin_id(self):
    """插件功能实现函数"""
    # 插件的主要功能代码
    pass
def README():
    """插件说明函数"""
    return [False, "插件名称和说明"]
```
### 要求

1. **文件命名**：使用英文小写字母和下划线，如 `my_plugin.py`
2. **主函数**：与文件名同名的函数，接收 `self` 参数
3. **README 函数**：返回插件说明信息
4. **代码注释**：建议添加必要的注释说明功能

## 🔧 故障排除

### 插件不工作？

1. 检查插件文件是否在正确的目录（`tool` 文件夹）
2. 确保语音助手已重启
3. 查看插件是否兼容当前版本

### 下载失败？

1. 尝试刷新页面
2. 清除浏览器缓存
3. 检查网络连接
4. 直接从 GitHub 下载
